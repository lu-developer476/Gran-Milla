
# Gran Milla

Proyecto web minimalista para exploración de destinos dentro de Argentina.

Tecnologías:
- React
- TypeScript
- Vite

Diseño:
- Interfaz blanca
- Tipografía azul estilo agencia de viajes
- Animaciones suaves de travesía

Incluye:
- Presentación extensa
- Lista de 10 destinos argentinos
- Formulario de contacto
- Variables de entorno listas para Vercel

## Deploy en Vercel

1. Subir el proyecto a GitHub
2. Importar el repositorio en Vercel
3. Configurar variables desde `.env`
